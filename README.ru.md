# WARP Deck

[English](README.md) | [Русский](README.ru.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | [العربية](README.ar.md) | [فارسی](README.fa.md) | [Türkçe](README.tr.md)

Плагин для Decky Loader на Steam Deck, который позволяет в один клик сгенерировать и подключиться к Cloudflare WARP через AmneziaWG (протокол с защитой от блокировок DPI), а также импортировать собственные конфигурации AmneziaWG/WireGuard.

> **Примечание:** Для корректной работы плагина на вашем устройстве должен быть обязательно установлен официальный клиент [Amnezia VPN](https://github.com/amnezia-vpn/amnezia-client).

![Screenshot](assets/screenshot.jpeg)

## 📋 Возможности

- **Cloudflare WARP в один клик**: Автоматическая генерация и обновление готового конфига WARP прямо из интерфейса плагина без ручной настройки.
- **Обход блокировок (AmneziaWG)**: Встроенная поддержка заголовков обфускации и мусорных пакетов AmneziaWG для обхода DPI блокировок провайдеров.
- **Импорт сторонних конфигов**: Возможность загрузить любой файл `.conf` (AmneziaWG или стандартный WireGuard).
- **Диагностика связи**: Встроенная проверка доступности хостов (1.1.1.1, Google и RuTracker).
- **Несколько профилей**: Удобное хранение и переключение между разными конфигурациями.
- **Всё включено**: Все необходимые бинарники AmneziaWG поставляются вместе с плагином и не требуют отдельной установки в систему.

> [!IMPORTANT]
> Встроенный генератор Cloudflare WARP автоматически настраивает все параметры обхода DPI (включая сигнатуру QUIC `I1`). Однако, если вы импортируете **сторонние VPN-профили**, их рекомендуется экспортировать из официального приложения [Amnezia VPN Client](https://github.com/amnezia-vpn/amnezia-client) в **«нативном формате AmneziaWG»** (файл `.conf` с параметрами `Jc`, `Jmin`, `Jmax`) для гарантированного обхода блокировок.

## 📥 Установка

1. Скачайте последнюю версию (`warp-deck.zip`) со страницы [Releases](https://github.com/rosakodu/warp-deck/releases).
2. Скопируйте ZIP-архив на ваш Steam Deck.
3. Включите **Режим разработчика** в системных настройках Steam Deck. Затем в настройках Decky Loader перейдите в раздел разработчика, активируйте **Developer mode** и выберите пункт «Установить плагин из файла» (или перетащите ZIP в область загрузки).

## 🚀 Использование

### 1. Быстрый запуск Cloudflare WARP (Рекомендуется)
После установки откройте плагин и нажмите кнопку **«Обновить»** (или плагин автоматически создаст профиль `warp-deck` при первом запуске). Будет создан рабочий конфиг с параметрами обфускации, который можно сразу включить тумблером.

### 2. Импорт сторонних VPN
Нажмите кнопку **«Импортировать конфиг»** и выберите файл `.conf` на вашем устройстве (например, из папки «Загрузки»).

## ⚙️ Как это устроено
Конфигурации плагина хранятся в папке пользователя `~/.local/share/warp-deck/configs`, а в системную папку `/etc/amnezia/amneziawg/` создаются символические ссылки. Управление интерфейсами происходит через встроенный модифицированный скрипт `awg-quick`.
